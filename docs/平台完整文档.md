## 1. 平台概述

### 1.1 平台定位
金猴火窝是一个专注于金融交易模拟挑战的平台，为用户提供：
- 模拟交易环境（MT4/MT5）
- 跟单学习系统
- 挑战赛系统
- 社区交流功能

### 1.2 核心功能模块

| 模块 | 说明 | 入口 |
|------|------|------|
| **大业** | 高级功能地图（开发中） | /home |
| **大赛** | 赛事汇总报名 | /lobby |
| **社交** | 聊天大厅 | /social |
| **空间** | 个人数据中心 | /user/space |
| **星球** | 跟单学习社区 | /planet |
| **EA商城** | EA产品购买 | /user/ea |
| **文档** | 帮助文档 | /docs |

### 1.3 货币系统
- **星球币**：平台虚拟货币
- 新用户注册赠送 **10,000 星球币**
- 用于报名K线征途（1000星球币/次）
- 用于购买EA产品
- 可充值获得

## 4. K线征途挑战赛

### 4.1 挑战概述

K线征途是一个10关闯关挑战赛，考验用户的持久盈利能力。

### 4.2 挑战规则

| 参数 | 值 |
|------|-----|
| 报名费 | 1,000 星球币 |
| 初始净值 | 1,000 |
| 通关目标 | 2,000 |
| 失败底线 | 100 |
| 挑战次数 | 无限（每次扣1000星球币）|

### 4.3 关卡配置

| 关卡 | 目标净值 | 失败底线 |
|------|----------|----------|
| 第1关 | 1,100 | 100 |
| 第2关 | 1,200 | 100 |
| 第3关 | 1,300 | 100 |
| 第4关 | 1,400 | 100 |
| 第5关 | 1,500 | 100 |
| 第6关 | 1,600 | 100 |
| 第7关 | 1,700 | 100 |
| 第8关 | 1,800 | 100 |
| 第9关 | 1,900 | 100 |
| 第10关 | 2,000 | 100 |

### 4.4 挑战流程

```
1. 点击"立即报名"
2. 扣除 1000 星球币
3. 申请进入"待审核"状态
4. 管理员审核 → 填写服务器、账号、密码
5. 审核通过 → 用户开始挑战
6. 账户净值达到目标 → 通关当前关卡
7. 继续挑战下一关
8. 通关第10关 → 挑战成功
9. 净值跌破100 → 挑战失败，可重新报名
```

### 4.5 挑战状态

| 状态 | 说明 |
|------|------|
| pending | 待审核 |
| approved | 已通过（待绑定账户）|
| active | 挑战进行中 |
| completed | 已通关 |
| failed | 挑战失败 |
| rejected | 申请被拒绝 |

### 4.6 交易规则

| 参数 | 值 |
|------|-----|
| 杠杆倍数 | 500倍 |
| 合约大小 | 100盎司/手 |
| 交易品种 | 伦敦金 |
| 报价来源 | 实时行情 |

### 4.7 名人堂

通关用户可进入名人堂，展示：
- 通关时间
- 总耗时
- 通关方式（匿名/公开）

## 5. 大赛汇总

### 5.1 赛事列表

| 赛事 | 周期 | 评判标准 | 入口 |
|------|------|----------|------|
| **天梯赛** | 每月赛季 | 当前赛季收益排名 | /lobby/ladder |
| **K线征途** | 单次挑战 | 10关通关 | /challenge/play |
| **大师邀请赛** | 季度 | 受邀参赛 | 邀请码 |
| **每日挑战赛** | 每日 | 当日收益率排名 | /lobby/daily |
| **月度总决赛** | 每月末 | 综合排名入围 | 自动入围 |
| **巅峰赛** | 季度 | 收益排名 | 资格认证 |

### 5.2 各赛事详情

#### 天梯赛
- **周期**：每月一个赛季
- **参赛费**：免费
- **评判**：当前赛季收益 + 总收益 排名
- **段位**：青铜 → 白银 → 黄金 → 铂金 → 钻石 → 王者

#### 大师邀请赛
- **参赛资格**：往届天梯赛冠军、月度总决赛前三
- **赛制**：淘汰制
- **奖励**：大师权杖勋章 + 高额奖金池分成

#### 每日挑战赛
- **报名时间**：每日 00:00 - 20:00
- **初始资金**：10,000 模拟金
- **结算时间**：次日 00:00
- **奖励**：前10名获得积分/京东卡

#### 月度总决赛
- **入围资格**：天梯赛 + 每日赛综合排名前200名
- **赛程**：每月27-30日
- **奖励**：黄金腰带 + 定制奖杯 + 赠金

#### 巅峰赛
- **参赛门槛**：天梯赛胜率 > 60%
- **初始资金**：50,000 模拟金
- **赛程**：季度一次
- **奖励**：保时捷 + 黄金

## 10. 数据库设计

### 10.1 核心表结构

#### 用户表 (user_accounts)
```sql
user_id        VARCHAR(36)     PRIMARY KEY
email          VARCHAR(255)    UNIQUE
password_hash  VARCHAR(255)
name           VARCHAR(100)    DEFAULT '用户'
role           ENUM           ('admin','user','vip')
coin_balance   DECIMAL(15,2)  DEFAULT 10000.00
avatar_url     VARCHAR(500)
created_at     TIMESTAMP
updated_at     TIMESTAMP
```

#### 星球表 (planets)
```sql
id             INT            PRIMARY KEY AUTO_INCREMENT
name           VARCHAR(255)
description    TEXT
rules          TEXT
creator_id     VARCHAR(255)   FK -> user_accounts
coins          INT            DEFAULT 0
ticket_price   INT            DEFAULT 0
invite_code    VARCHAR(50)
max_publishers INT            DEFAULT 3
status         ENUM          ('active','closed')
duration_days  INT            DEFAULT 365
expire_at      TIMESTAMP
```

#### 星球成员表 (planet_members)
```sql
id             INT            PRIMARY KEY AUTO_INCREMENT
planet_id      INT            FK -> planets
user_id        VARCHAR(255)   FK -> user_accounts
role           ENUM          ('owner','publisher','follower')
join_method    ENUM          ('purchase','invite')
ticket_paid    INT            DEFAULT 0
joined_at      TIMESTAMP
expiry_date    TIMESTAMP
```

#### 信号表 (signals)
```sql
id             INT            PRIMARY KEY AUTO_INCREMENT
created_at     TIMESTAMP
sender_account VARCHAR(255)
signal_type    VARCHAR(50)    -- buy/sell
ticket         INT
symbol         VARCHAR(50)    -- XAUUSD
order_type     VARCHAR(10)    -- market/limit
volume         DECIMAL(10,2)
price          DECIMAL(10,4)
sl             DECIMAL(10,4)  -- 止损
tp             DECIMAL(10,4)  -- 止盈
comment        TEXT
deal_profit    DECIMAL(10,2)
balance        DECIMAL(10,2)
broker         VARCHAR(255)
planet_id      INT            FK -> planets
```

#### MT账户表 (mt_accounts)
```sql
id             INT            PRIMARY KEY AUTO_INCREMENT
user_id        VARCHAR(255)   FK -> user_accounts
account_number VARCHAR(50)
broker         VARCHAR(255)
platform       ENUM          ('MT4','MT5')
is_verified    BOOLEAN        DEFAULT false
verified_at    TIMESTAMP
created_at     TIMESTAMP
```

#### K线征途报名表 (challenge_registrations)
```sql
id             INT            PRIMARY KEY AUTO_INCREMENT
user_id        VARCHAR(255)   FK -> user_accounts
status         ENUM          ('pending','approved','rejected','active','completed','failed')
current_level  INT            DEFAULT 1
completed_levels TEXT          -- JSON: [1,2,3...]
started_at     TIMESTAMP
completed_at   TIMESTAMP
failed_at      TIMESTAMP
failed_level   INT
total_duration INT            -- 总耗时（秒）
server_name    VARCHAR(255)   -- 管理员填写
trading_account VARCHAR(50)
trading_password VARCHAR(255)
mt_account_id  INT            FK -> mt_accounts
```

#### K线征途关卡配置表 (challenge_level_config)
```sql
id             INT            PRIMARY KEY AUTO_INCREMENT
level          INT            UNIQUE 1-10
name           VARCHAR(100)
description    TEXT
target_balance INT            DEFAULT 2000
initial_balance INT           DEFAULT 1000
fail_balance   INT            DEFAULT 100
reward         VARCHAR(255)
is_active      BOOLEAN        DEFAULT true
```

#### EA产品表 (ea_products)
```sql
id             INT            PRIMARY KEY AUTO_INCREMENT
name           VARCHAR(255)
description    TEXT
price          INT            -- 星球币
version        VARCHAR(50)   DEFAULT '1.0.0'
platform       ENUM          ('MT4','MT5','Both')
category       VARCHAR(100)  -- 趋势/震荡/马丁
product_type   VARCHAR(50)   -- ea/indicator/script/tool
features       TEXT           -- JSON
download_url   VARCHAR(500)
file_name      VARCHAR(255)
file_size      INT            -- KB
image_url      VARCHAR(500)
images         TEXT           -- JSON数组
status         ENUM          ('active','inactive')
creator_id     VARCHAR(255)
sales_count    INT            DEFAULT 0
```

#### 系统配置表 (system_config)
```sql
id             INT            PRIMARY KEY AUTO_INCREMENT
config_key     VARCHAR(100)   UNIQUE
config_value   TEXT
description    VARCHAR(255)
```

#### 论坛帖子表 (forum_posts)
```sql
id             INT            PRIMARY KEY AUTO_INCREMENT
planet_id      INT            FK -> planets
user_id        VARCHAR(255)   FK -> user_accounts
title          VARCHAR(255)
content        TEXT
like_count     INT            DEFAULT 0
comment_count  INT            DEFAULT 0
is_pinned      BOOLEAN        DEFAULT false
status         ENUM          ('active','hidden','deleted')
```

### 10.2 索引设计

| 表名 | 索引类型 | 字段 |
|------|----------|------|
| planets | INDEX | creator_id |
| planet_members | UNIQUE | planet_id + user_id |
| signals | INDEX | sender_account |
| signals | INDEX | planet_id |
| forum_posts | INDEX | planet_id, user_id, status |
