## 1. 项目概述

### 1.1 项目简介

**项目名称**：喵了个喵 - 星球跟单平台

**项目定位**：一个专业的MT4/MT5外汇交易信号跟单平台，连接信号发布者与跟单者，实现交易信号的实时共享与智能跟单。

**核心价值**：
- 为信号发布者提供信号分发渠道
- 为跟单者提供专业、稳定的跟单服务
- 通过"星球"概念组织社区，形成交易者生态圈

### 1.2 产品特色

| 特色 | 说明 |
|------|------|
| 星球社区 | 以星球为单位组织交易社区，星主可管理信号发布者和跟单者 |
| 智能跟单 | 支持一键跟单、灵活控制跟单参数 |
| 信号管理 | 交易信号实时推送，支持历史查询 |
| EA集成 | 提供EA产品下载，支持自动化交易 |
| K线征途 | 交易挑战赛功能，10关闯关挑战 |
| 文档中心 | 完善的新手引导和功能文档 |

## 2. 目标用户

### 2.1 用户角色

| 角色 | 描述 | 核心需求 |
|------|------|----------|
| **信号发布者 (Publisher)** | 专业交易员，通过EA或手动发布交易信号 | 发布信号、管理跟随者、查看收益统计 |
| **跟单者 (Follower)** | 普通交易者，复制跟随信号交易 | 浏览星球、订阅信号、设置跟单参数 |
| **星主 (Owner)** | 星球的创建者和管理员 | 管理星球成员、审核申请、设置门票价格 |
| **普通用户 (User)** | 注册但未加入星球的用户 | 浏览星球、注册登录、了解平台 |
| **管理员 (Admin)** | 平台运营人员 | 系统配置、用户管理、内容审核、K线征途审核 |

### 2.2 用户画像

**信号发布者**：
- 有稳定的外汇交易策略
- 希望将策略规模化，产生额外收益
- 具备一定的技术能力，能配置EA

**跟单者**：
- 缺乏交易时间或经验
- 希望复制专业交易员的策略
- 追求稳定收益，控制风险

### 4.6 K线征途

#### 4.6.1 挑战赛概述

**概念定义**：
- K线征途是一个交易挑战赛
- 共10个关卡，每个关卡有目标净值
- 初始净值1000，失败底线100，通关目标2000
- 用户需支付报名费参与

**关卡配置**：

| 3 | 守戒 | 2000 | 100 |
| 4 | 忍痛 | 2000 | 100 |
| 5 | 止喜 | 2000 | 100 |
| 6 | 观己 | 2000 | 100 |
| 7 | 破执 | 2000 | 100 |
| 8 | 随势 | 2000 | 100 |
| 9 | 忘我 | 2000 | 100 |
| 10 | 得道 | 2000 | 100 |

#### 4.6.2 挑战流程

**流程图**：
```
用户报名 → 扣除报名费 → 待审核 → 管理员审核 → 分配账号 → 激活挑战 → 挑战进行中 → 通关/失败 → 可重新报名
```

**状态说明**：
| 状态 | 说明 |
|------|------|
| pending | 待审核 |
| approved | 已通过，待激活 |
| active | 挑战进行中 |
| completed | 已通关 |
| failed | 挑战失败 |
| rejected | 申请被拒绝 |

**功能点**：
| 功能 | 描述 | 状态 |
