金火火 - 智能交易平台

方式一：添加到主屏幕（推荐）
无需下载安装，像App一样使用
1

打开网站

用 Chrome 或 Safari 浏览器打开：

2

点击分享按钮

点击浏览器底部的"分享"按钮（或菜单中的"添加到主屏幕"）

3

添加到主屏幕

选择"添加到主屏幕"或"安装应用"

4

完成

桌面上会出现"金火火"图标，点击即可使用

方式二：构建 APK 安装包
需要在电脑上使用 Android Studio 构建
说明
服务器环境无法直接生成APK。请下载项目源码，在您自己的电脑上使用 Android Studio 构建 APK 安装包。

构建步骤：

1

安装 Android Studio

从官网下载并安装 Android Studio (最新版本)

https://developer.android.com/studio
2

下载项目源码

点击上方按钮下载项目压缩包，解压到本地目录

tar -xzvf jinhuohuo-android-app.tar.gz
3

用 Android Studio 打开项目

打开 Android Studio，选择 "Open an existing project"，选择解压后的 android-app 目录

4

等待 Gradle 同步

首次打开会自动下载依赖，请耐心等待（可能需要5-10分钟）

5

构建 APK

点击菜单 Build > Build Bundle(s) / APK(s) > Build APK(s)

6

获取 APK 文件

构建完成后，APK 位于 app/build/outputs/apk/debug/

app/build/outputs/apk/debug/MiaoLeGeMiao_1.0.0_debug_*.apk
